레크레이션 진행 화면 🎛️

이 노트북 화면을 빔에 연결해 다 같이 보는 진행 화면으로 쓰세요. 진행은 버튼 또는 Space 로 합니다.

💡 진행 방식 — 각 게임에서 가족들이 폰으로 답을 제출 → 진행자가 답안 공개 → 맞힌 가족에게 점수. (외쳐서 맞히기도 가능)
참가 가족 설정

가족 이름을 직접 바꿀 수 있어요(가족 폰에서도 가능). 여기서 만든 가족이 모든 게임·점수판에 그대로 쓰여요.

SCORE BREAKDOWN

게임별 점수 집계 📊

가족별로 ①~④ 게임에서 받은 점수와 합계를 한눈에. 칸의 −/+ 로 직접 보정할 수 있어요.

합계 = ①장소 + ②인물 + ③요리 + ④소개 + 조정. 새로고침해도 점수는 유지돼요(localStorage).

CORNER 01 · SETUP

교회 장소 등록 📍

맞출 장소 이름을 쓰고, 노트북에 있는 사진을 올려주세요. 첫 항목은 연습용 예시(사진 자동 내장)예요. (업로드한 사진은 이 세션에서만 유지 — 행사 직전 등록 추천)

💡 줌인 추천 스팟: 본당 십자가 디테일 · 카페 · 기도실 · 친교실 · 주차장 표지판 · 게시판 · 교역자실 문패 · (웃음용) 교역자 머그컵. 사진은 핵심 부분만 클로즈업해 찍으면 줌아웃 재미가 커져요.

맞히는 법 — 줌아웃하며 보여주다가, 아래 답안 보드로 가족 폰 답을 받아 답안 공개 → 맞힌 가족 지급. (외쳐서 맞히기도 가능 — 단 한 게임에 한 방법만!)

CORNER 05 · 보너스

순위 예측 보너스 🔮

게임 시작 전, 각 가족이 ①~④ 게임의 우승 가족을 예측해요. 행사 끝에 적중한 개수만큼 보너스 점수를 받습니다!

진행 — (게임 전) 각 가족이 폰으로 ①~④ 우승 예측을 제출(또는 표에 직접 입력) → (모든 게임 후) 결과 채점을 누르면 실제 우승 가족과 비교해 보너스가 ⑤칸에 자동 기록돼요. 다시 눌러도 안전(덮어쓰기).
예측 입력 — 각 가족이 찍은 '게임별 우승 가족'
적중 1개당

각 가족이 'OO게임은 △△가족 우승!'이라고 예상한 가족을 고르면 자동 저장돼요. 안 찍은 칸은 '—'로 두면 됩니다.
